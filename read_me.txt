This is a repository of the Face Films project for the IOT module created by Tomáš Kňaze

To see the code for the data collection/film watching go to ---> Data_Collect_Code
  It contains the collect_main.py file which imports functions from the functions.py file
  The Emotion_weights.h5 is the neural network model for emotion recognition it is created using classification.py and also imported into collect_main.py




To test out the data collecting app, either download the files from Data_Collect_Code, install modules from requirements.txt and run collect_main.py or download the executable here DROPBOX
